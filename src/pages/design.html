Archive: A Design Library Mobile App

As a student, it can be frustrating to have all of your resources spread out across multiple platforms. This app aims to solve that problem by providing one location for design students to go to access library of all the materials they could need. Through this project, I learned a lot about Figma variables and was able to create a functional light/dark mode.

Community Services of Moses Lake

Recently, I have been working on a website redesign for an active food bank in Moses Lake, WA. Their current site is hard to navigate and doesn't work well on mobile devices. This new design aims to make navigation simple and straightforward so patrons can easily find the information they are searching for. My goal with the next iteration of this design is to improve the design so users feel a more personal connection to the organization through the site.

Smart Mirror

The first project I ever completed in Figma was designing a smart mirror UI for another student in my UX class. I learned a lot about how to use the software through the course of this project, including how to animate text. The final design of this product was polished through several iterations and feedback sessions. Although it is simple, this project was important to my growth as a UX designer and it helped me to build the foundation needed to be able to progress to where I am today.
